# Clickable Sidebar Review Surfaces

GitHub issue: `#29`

## Parent

GitHub issue `#27` - PRD: MVP+3 Gmail sidebar interactive teaching loop

## What to build

Make sidebar review surfaces useful instead of inert. Today metric tiles should open matching review lists, and "What changed today" should group by outcome/action with clickable items that route to useful email/review state.

## Acceptance criteria

- [ ] Today metrics such as processed, auto-handled, and kept visible open matching sidebar review lists.
- [ ] "What changed today" is grouped by action/outcome and sorted by recency within groups.
- [ ] Items in review lists are clickable and open/select the related email or usable review state where reliable.
- [ ] Tests cover the grouping and click behavior without requiring live Gmail.
