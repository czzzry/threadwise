Threadwise × PostHog architecture sheet

The compressed mental model: PostHog is a privacy-gated, non-authoritative observer attached to the existing extension → local companion boundary.

The path

User transition
  → analytics.js chooses approved event + safe properties
  → extension background worker attaches anonymous installation ID
  → POST http://127.0.0.1:8021/api/analytics/capture
  → ProductAnalytics validates again
  → Python PostHog SDK
  → https://eu.i.posthog.com

Any analytics failure → false / dropped event → core product continues
